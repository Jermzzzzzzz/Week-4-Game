Crystal Collector Game

Instructions


  1. You will be given a random number at the start of the game."Make your exactly match this number exactly."
  2. There are four crystals below.
    • Blue
    • Green
    • Red
    • Yellow
  3. Clicking on each crystal will add a different amount of points to your total score.
  4. Win the game by matching your score to the number given to you.
  5. If your score goes above the number you are given YOU LOSE!
  6. The value of each crystal is not shown, figure it out.
  7. Each time when the game starts, the game will change the values of each crystal.
